# Stygian Blue

This is my custom [bootc](https://github.com/bootc-dev/bootc) / [Universal Blue](https://github.com/lbussell/stygian-blue.git) OS image.

## Installation

If you aren't me you probably don't want to do this.

1. Install Fedora Kinoite or Bazzite (KDE version).
2. Rebase onto this image: `sudo bootc switch ghcr.io/lbussell/stygian-blue`
